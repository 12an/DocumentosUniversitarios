Herramientas Informáticas  para Ingenieros              con Microsoft® Excel                   y Microsoft® Project              

Es la intención de este libro, poner al alcance de los Ingenieros, las herramientas informáticas para el desarrollo de su actividad profesional. Nos referimos a: Microsoft® Excel, Microsoft® Access, Microsoft® Project, Microsoft® Visio, todos productos con una potencialidad inigualable para la realización de cálculos, análisis, seguimientos, gestión y presupuestación. En este sentido, los autores han realizado una combinación de todos ellos, pensando en la especificidad de la Ingeniería.

            Herramientas Informáticas para Ingenieros          con Microsoft® Excel y Microsoft® Project 
Contenido

Herramientas Informáticas para Ingenieros, está desarrollado para que los profesionales, como usuarios de estos productos, encuentren rápidamente ideas y soluciones sobre estas herramientas.

El libro posee un capítulo destinado a la explicación exhaustiva de las herramientas más usadas dentro de Microsoft® Excel 2003 y Microsoft® Project 2002, para que el ingeniero aprenda a utilizar estos programas de tanta utilidad y versatilidad, y de esta forma pueda ahorrar tiempo y optimizar su trabajo cotidiano.

Los autores presentan diferentes planillas prediseñadas listas para usar en Microsoft® Excel, Microsoft® Project, Microsoft® Access y Microsoft® Visio, incluidas en un CD, con sus respectivos manuales de uso explicados en el libro.

Asimismo, las planillas se organizan en cuatro grandes secciones que se detallan a continuación:

SOLUCIONES INCLUIDAS

PLANILLAS PARA EL ANÁLISIS:

                 · Análisis ABC de Costos.xls

                 · Gestión de Proyectos y Empleados.mdb

                 · Análisis de Inversión.xls

                 · Retorno de Inversión en Tecnología.xls

PLANILLAS PARA LA GESTIÓN:

                 · Inventario.xls 

                 · Gestión Mano de Obra.xls  

                 · Nota de Pedido.xls 

                 · Ficha de Proyecto.xls

                 · Entrada de Materiales.xl

PLANILLAS PARA EL SEGUIMIENTO:

                 · Avance de Proyecto.xls   

                 · Etapas para el Desarrollo de Software (Project). 

                 · Implantación de un nuevo Sistema en la Empresa (Project) 

                 · Topología de Redes (Visio)

PLANILLAS PARA EL PRESUPUESTO:

                 · Cálculo y Presupuesto.xls

                 · Presupuesto Simple.xls

 

Deseamos que a través de este libro podamos acercarles, de una forma didáctica y completa, herramientas e ideas para la aplicación de los distintos productos de Microsoft® Corp. en las tareas diarias del ingeniero.

Requerimentos: Memoria RAM mínima de 64 MB, Microsoft® Office 2000 o superior, Microsoft® Project 2000 o superior, Microsoft® Visio 2000 o superior; Sistema operativo Windows 98 (2da Edición) o superior, Windows NT versión 4.0 o superior, Windows XP o superior.
