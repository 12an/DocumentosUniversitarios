Ventas y Marketing     con Microsoft Excel

El objetivo fundamental de este trabajo es brindar al lector ciertos conceptos esenciales en temas relacionados con Ventas y Marketing, poniendo a su disposición, una serie de prácticos aplicativos informáticos de sencilla e inmediata utilización, desarrollados especialmente para cada tópico abordado.

El vertiginoso avance del mundo comercial de hoy exige, a quienes desean ser parte del competitivo escenario empresarial, actualizarse y perfeccionarse permanentemente en materia informática y conceptual.

 Ventas y Marketing con Excel
Contenido

El contenido de este libro le permitirá articular las nociones introductorias de cada capítulo con las plantillas prediseñadas en Microsoft® Excel. De esta manera, y guiándose con los Manuales de Uso correspondientes a cada aplicación aquí incluidos, podrá complementar conocimientos y mejorar habilidades para la práctica cotidiana.

 

SOLUCIONES INCLUIDAS

Entre otros, los aplicativos abordados son:

·         Análisis de Ventas

·         Encuestas

·         Análisis Financiero de Propuestas Comerciales

·         Comisiones de Vendedores

·         Campañas en Medios de Comunicación

·         Armado y Control de Presupuestos

·         Seguimiento de Clientes

·         Logística: Desviación por Cantidad y Canal

·         Lista de Precios - Cotizador

Finalmente, la herramienta propuesta para facilitar nuestras tareas, se basa en el estudio con las planillas de cálculo de Microsoft® Excel. Las herramientas utilizadas y explicadas en este libro son: Filtros automáticos de información mediante diversos criterios, resumen de información con tablas dinámicas y formato condicional, entre otras.

Este trabajo está orientado para quien esté emprendiendo un proyecto comercial; para el empresario PyME (pequeñas y medianas empresas); y para todo aquel que desee disponer de ciertas soluciones informáticas y acercarse a los conocimientos básicos de estas dos disciplinas, tan protagonistas en el ámbito empresario del mundo de hoy, las Ventas y el Marketing.

Utilice este libro para comenzar a comprender que, para emprendedores y pequeños empresarios, también existen oportunidades de llevar a cabo exitosas acciones comerciales.

 

Requerimientos: Memoria RAM mínima de 64 MB; Microsoft Office 2000 o superior; sistema operativo: Microsoft Windows 98 (2da edición) o superior, Windows NT versión 4.0 o superior, Windows XP o superior.
