Conceptos Fundamentales        Windows Server System                        

Con la línea 2003 de sus productos, Microsoft® replanteó su visión y estrategia para brindar a sus clientes soluciones que reduzcan la complejidad y, al mismo tiempo, incorporen innovaciones en una infraestructura totalmente integrada. De esta manera, ha unificado su gama de productos de servidor bajo una única denominación: Windows Server System

Estas herramientas informáticas ayudarán a cuantificar, medir y evaluar el impacto que tiene la inflación en los negocios. Dichas aplicaciones en Microsoft® Excel ofrecidas, se encuentran con sus respectivos manuales de uso incluidos en el libro.

Orientado a quienes deben tomar decisiones en el área de sistemas (gerentes, consultores, jefes de proyecto, etc.) este libro tiene el fin de dilucidar, paso a paso, en qué consiste Windows Server System, detallando cada una de las áreas en las que se divide y explicando sus principales funcionalidades, su correcto licenciamiento, y los costos de los productos involucrados:

Infraestructura de IT: Microsoft® Internet Security and Acceleration (ISA) Server 2000, Systems Management Server (SMS) 2003, Microsoft® Operations Manager (MOM) 2000, Microsoft® Application Center 2000, Windows® Storage Server 2003, Windows® Small Business Server.

Infraestructura de Aplicaciones: Microsoft® Visual Studio®.NET, Microsoft® SQL Server™ 2000, Microsoft® BizTalk Server 2002, Microsoft® Commerce Server 2002, Microsoft® Content Management Server 2002, Microsoft® Host Integration Server 2000, Microsoft® Speech Server 2004, Microsoft® Identity Integration Server 2003.

Infraestructura para Information Workers: Microsoft® Exchange Server 2003, Microsoft® SharePoint Portal Server 2003, Microsoft® Office Live Communications Server 2003, Microsoft® Office Project Server 2003.

Todos montados sobre una sólida Plataforma de Servidor: Windows Server 2003.

El objetivo final es, en definitiva, brindar a los profesionales de IT toda la información necesaria para que, sabiendo lo que hoy existe y lo que habrá en un futuro, puedan tomar las mejores decisiones sobre tecnología.
