Excel para PyMEs 2

El objetivo que se desea alcanzar con este trabajo, es la utilización de una de las herramientas informátcias más importante que poseen la mayoría de las Empresas, Microsoft Excel para agilizar la operatoria normal de las PyMEs.
Excel para PyMEs 2, lo que persigue es que Usted, tanto como Empresario o como usuario de los programas, encuentre en este libro, soluciones a las tareas que hoy en día ocupan la mayor parte del tiempo en el área de Gestión, Control de la Gestión en la Empresa.
