Microsoft Excel para Contadores

El objetivo que se desea alcanzar con este trabajo, es la utilización de una de las herramientas informáticas más importantes que poseen la mayoría de los profesionales en Ciencias Económicas, Microsoft Excel, un producto de la gama Office con una potencialidad inigualable para realizar cálculos y análisis de datos.
Microsoft Excel para Contadores, está pensado para que usted como usuario del producto, encuentre rápidamente ideas y soluciones presentadas sobre esta herramienta.
