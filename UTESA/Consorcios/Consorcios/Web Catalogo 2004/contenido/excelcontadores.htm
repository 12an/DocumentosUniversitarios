Microsoft® Excel para Contadores

El objetivo que se intenta alcanzar en este trabajo, es la utilización de una de las herramientas más importantes que poseen la mayoría de los profesionales en ciencias económicas, Microsoft® Excel, un producto de la gama Office con una potencialidad inigualable para realizar cálculos y análisis de datos. 
Microsoft® Excel para contadores, está pensado para que usted como usuario del producto, encuentre rápidamente ideas y soluciones presentadas sobre esta herramienta.

Microsoft® Excel para Contadores
Herramientas Informáticas para la Pequeña y Mediana Empresa
Contenido:
Este libro se divide en cinco grandes secciones en las cuales se presentan diferentes plantillas prediseñadas en Microsoft® Excel para facilitar la gestión en las empresa
 
Análisis
Planillas diseñadas para análisis de una empresa.
  • Análisis de Resultados
  • Análisis de Empresa
  • Análisis Patrimonial para Empresas
  • Análisis ABC de Ventas
  • Índices y Rátios
Administración
Planillas pensadas para que el administrador pueda realizar cálculos y controles.
  • Cálculo de Punto de Equilibrio
  • Cancelación de Comprobantes
  • Cartera de Valores
  • Costo Total, Medio y Marginal
Presupuesto
Análisis, confección y desviaciones de presupuestos
  • Presupuesto de Ventas y Compras
  • Cash Flow
  • Desviación de Presupuesto de Ventas
Contables
Planillas de uso contable para facilitar el manejo de la contabilidad.
  • Anexo de Balance
  • EVCC
  • EOAF
  • Libro Diario y Mayores
Organizativas
Las planillas de esta área sirven para oganizar las tareas cotidianas de cualquier empresa o profesional.
  • Libro Diario de Caja
  • Libro de Banco
  • Administración de Curriculums Vitae
