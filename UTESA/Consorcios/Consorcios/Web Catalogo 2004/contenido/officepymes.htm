Office para Pymes

  Aplicaciones en Excel y Word para la 
  Pequeña y Mediana Empresa

El objetivo de este libro es implementar la potencialidad de Microsoft® Office en cualquier tipo de Empresa.  Para ello se presentan plantillas listas para usar realizadas en Microsoft® Excel y Word.

Se encuentra en forma detallada el uso de cada una de estas plantillas, para que cualquier topo de usuario de los productos Microsoft® puede utilizarlas sin inconveniente alguno, sepa o no el uso de cada uno de estos productos.

 


 
 

Los autores han descubierto que la mayoría de las empresas poseen de alguna manera los productos Microsoft® pero no todas ellas obtienen la máxima potencialidad.
Este trabajo ayuda en cierta forma a mejorar la relación usuario de Pc´s y software preinstalado.

El alcance de este libro, no se limita a un sector o mercado, sino que está preparado para ser utilizado por todo tipo de empresa PyMe.
