    CURSO DE SERVIDORES DE INTERNET EN WINDOWS

      

          

    Continuando con los complementos para nuestro Servidor, esta vez nos vamos a dedicar a instalar un Gestor de Bases de Datos Relacionales. En este caso hemos elegido MySQL, ya que es el mas popular gestor de bases de datos SQL y además, como no, gratuito (es Open Source)    :))))))

    La primera tarea a realizar es la de bajarnos el programa. Vamos a bajarnos en el mismo archivo tanto el Servidor como el cliente. Pulsa aquí. Son algo más de 9 Mb, así que tardarás un poco :(

    Ahora vamos a instalarlo. Es un archivo comprimido con ZIP, con lo que tendrás que descomprimirlo en una carpeta para ejecutar la instalación.

    1º- Abre la carpeta donde has descomprimido el fichero, y pulsa sobre el ejecutable Setup.exe para iniciar la Instalación. La primera pantalla es de Bienvenida y como siempre recomendando que cierres todos los programas que estes ejecutando, por si acaso.  Pulsa el botón  Next .

    2º Ahora viene una pantalla con información de la verison que estamos instalando y con una observación para los que usais NT y queréis instalar MySQL como servicio (solo debeis crear un fichero llamado my.cnf con dos líneas que son [mysqld] y basedir=C:/mysql/ ). Pulsa el botón  Next

    3º Nos pide confirmación del directorio de la instalación que por defecto es C:\mysql Si quereis cambiarlo pulsáis browse y elegís donde.. Pulsa el botón  Next 

    4º Ahora podemos elegir el tipo de instalación que queremos. Vamos como siempre a instalar la típica. Pulsar   Next  

    5º Pues ya está. Ha terminado la instalación, y el programa ya se ha instalado en el directorio c:\mysql. Pero, atención, que no instala ningún acceso directo en el Menu Programas (tendréis que ponerlo vosotros). Tenemos además una aplicación para manejar MySQL. Esta aplicación se encuentra en c:\mysql\bin\winmysqladmin.exe (podéis ejecutarla desde MSDOS o desde el explorador de Windows, pulsando dos veces sobre el fichero winmysqladmin.exe ). La primera vez nos va a solicitar los datos de usuario y Password para crear el fichero my.ini (se instala en el directorio c:\windows)

    6º El programa winmysqladmin os ayudará a mantener y controlar el funcionamiento de mysql. Podeis incluir en vuestro menú de inicio éste programa para poner en marcha o parar mysql.

    7º Para arrancar MySQL tenemos tres ficheros posibles (dentro del directorio c:\mysql\bin\: mysqld.exe que es el genérico de los sistemas 386 en adelante. mysqld-opt.exe para equipos pentium y mysqld-nt.exe para los basados en NT. Para detener MySQL lo haremos mediante mysqlshutdown.exe (acordaros de hacerlo antes de apagar el ordenador)

    La forma de crear una base de datos es con la orden

     mysqladmin -u root create nombrebasededatos

    Ahora ya podéis disfrutar de éste magnifico gestor de Bases de Datos. Si no lo conocéis, ya es hora de que empecéis a aprenderlo, así que a  buscar en Internet que encontrareis mas de un Manual)
