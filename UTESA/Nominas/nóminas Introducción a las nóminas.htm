nóminas - Lección 1: Introducción a las nóminas

Este curso está destinado al estudio de las nóminas. Este documento es un recibo que justifuca el pago del salario al trabajador. Las nóminas también se conocen con el nombre de recibo de salarios.

A lo largo de este curso vamos a estudiar exhaustivamente la nómina (recibo de salarios) y cada una de las características que conforman este documento laboral.

El salario es toda percepción económica que reciba el trabajador. Se puede recibir el salario en dinero o en especie. En este último caso nunca puede ser superior al 30% de las percepciones salariales.
